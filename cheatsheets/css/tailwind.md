# Tailwind CSS Cheat Sheet

## Layout & Box Model
- `container`: Center content with max-width.
- `block`, `inline`, `flex`, `grid`, `hidden`.
- `p-{x}`, `m-{x}`: Padding and margin (e.g., `p-4`, `mx-auto`, `mt-2`).
- `w-{size}`, `h-{size}`: Width and height (e.g., `w-full`, `h-screen`, `w-1/2`).
- `max-w-{size}`: `max-w-md`, `max-w-xl`, `max-w-7xl`.

## Flexbox & Grid
- `flex`, `flex-col`, `flex-row`.
- `justify-start`, `justify-center`, `justify-between`.
- `items-center`, `items-start`, `items-end`.
- `grid`, `grid-cols-1`, `grid-cols-3`.
- `gap-4`, `gap-x-2`, `gap-y-8`.

## Typography
- `text-xs`, `text-sm`, `text-base`, `text-lg`, `text-xl`, `text-2xl`.
- `font-bold`, `font-semibold`, `font-light`.
- `text-center`, `text-left`, `italic`.
- `text-blue-500`, `text-gray-900`.

## Interactivity & Transitions
- `hover:bg-blue-700`.
- `focus:outline-none`.
- `transition`, `duration-300`, `ease-in-out`.
- `cursor-pointer`, `select-none`.

## Responsive Design
- `sm:` (640px), `md:` (768px), `lg:` (1024px), `xl:` (1280px).
- Example: `className="w-full md:w-1/2"`
